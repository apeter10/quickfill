核心功能

簡單三步驟，從此告別重複填寫

📋

一鍵擷取表單

填好表單後，點擊 QuickFill 按鈕即可擷取所有欄位值。系統自動辨識每個欄位的定位器，確保精準對應。支援 input、select、textarea、radio、checkbox 等各種表單元素。

⚡

自動填寫

下次造訪相同頁面時，QuickFill 自動偵測匹配的規則並填入表單。支援 React、Vue 等前端框架的受控元件，確保填寫後表單狀態正確更新。

📂

規則管理

在管理頁面查看所有規則，按網站分組顯示。支援編輯欄位值、重新命名規則、刪除單條或批次刪除。為不同家庭成員各存一組資料也沒問題。

🌐

智慧 URL 比對

規則依據網站的 Domain + Path 綁定，忽略 Query 參數變化。掛號頁面不管日期怎麼變，規則都能正確匹配到相同的表單。

🔄

跨裝置同步

將重要規則標記後，可選擇性同步至 Chrome 帳號。在家裡電腦建立的規則，到公司的電腦也能使用。完全透過 Chrome 原生同步機制，無需額外帳號。

🔒

安全至上

絕不擷取密碼欄位與信用卡資訊。所有資料僅儲存在瀏覽器本地，不傳送至任何外部伺服器。你的個人資料，只有你自己能看到。

安裝方式

選擇適合你的安裝方式

🏭 Chrome Web Store

  1. 點擊上方「安裝 Chrome 擴充功能」按鈕
  2. 在 Chrome Web Store 頁面點擊「新增至 Chrome」
  3. 確認權限後即可開始使用

💻 開發者模式安裝

  1. 下載並解壓縮原始碼
  2. 開啟 Chrome，前往 chrome://extensions/
  3. 開啟右上角「開發人員模式」
  4. 點擊「載入未封裝項目」，選擇 quickfill/ 資料夾

隱私權政策

最後更新日期：2026 年 4 月 1 日

資料收集

QuickFill 不收集、不傳輸、不分享任何使用者的個人資料。擴充功能運作完全在使用者的瀏覽器本地端進行。

資料儲存

使用者透過 QuickFill 擷取的表單資料，僅儲存於以下位置：

  • chrome.storage.local — 瀏覽器本地端儲存，僅限於使用者自身裝置
  • chrome.storage.sync（選擇性啟用）— 透過使用者自身的 Google 帳號同步至其他裝置，由 Chrome 瀏覽器原生加密傳輸

除上述兩者外，QuickFill 不會將資料傳送至任何外部伺服器、第三方服務或分析平台。

敏感資料處理

QuickFill 在擷取表單時，會自動排除以下類型的欄位：

  • 密碼欄位（type="password"）
  • 含有信用卡號、CVV、安全碼等敏感關鍵字的欄位

上述欄位的值不會被讀取、不會被儲存。

權限說明

QuickFill 所需的 Chrome 擴充功能權限及其用途：

  • storage — 用於將規則資料儲存於瀏覽器本地端
  • activeTab — 用於存取當前分頁的頁面內容以進行表單擷取與填寫
  • scripting — 用於在頁面中執行表單掃描與填寫腳本
  • webNavigation — 用於偵測頁面框架結構，以支援含有 iframe 的表單頁面

第三方服務

QuickFill 不使用任何第三方分析工具（如 Google Analytics）、廣告服務或資料追蹤器。擴充功能不包含任何會向外部發送請求的程式碼。

資料刪除

使用者可隨時透過擴充功能的管理頁面刪除任何已儲存的規則。移除（解除安裝）擴充功能後，所有儲存於 chrome.storage.local 的資料將被瀏覽器自動清除。

兒童隱私

QuickFill 不針對兒童設計，也不會刻意收集任何兒童的個人資訊。

政策變更

若本隱私權政策有所變更，將在此頁面更新內容與日期。重大變更將透過擴充功能內的通知告知使用者。
